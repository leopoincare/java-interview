## 数组和链表在 Java 中的区别是什么？

这是 **Java 数据结构面试高频题**，核心考察 **底层实现、访问效率和适用场景**。我帮你整理成 **区别 → 优缺点 → 面试总结**。

## 面试总结一句话

> **数组在 Java 中是连续内存、支持快速随机访问但插入删除慢；链表是节点引用连接、插入删除快但随机访问慢，选择哪种结构取决于访问模式和操作频繁度。**

------

## 一、数组（Array）和链表（LinkedList）的核心区别

| 特性           | 数组（Array）                        | 链表（LinkedList）                   |
| -------------- | ------------------------------------ | ------------------------------------ |
| **底层结构**   | 连续内存空间                         | 节点通过引用连接（链式结构）         |
| **随机访问**   | O(1)，通过索引直接访问               | O(n)，需从头遍历                     |
| **插入/删除**  | O(n)，需要移动元素                   | O(1)（已定位节点），无需移动其他元素 |
| **内存占用**   | 紧凑，每个元素只存数据               | 每个节点额外存储指针（prev/next）    |
| **大小可变性** | 固定大小（普通数组），动态数组可扩容 | 可动态增长，无需连续内存             |
| **缓存友好性** | 高，连续内存利于 CPU 缓存            | 低，分散内存访问慢                   |

------

## 二、Java 中实现

### 1️⃣ 数组

```
int[] arr = new int[10];        // 原生数组
Integer[] arrObj = new Integer[10]; 
```

### 2️⃣ 链表

- Java 提供 `LinkedList` 类（双向链表）

```
LinkedList<Integer> list = new LinkedList<>();
list.add(1);      // 添加元素
list.add(0, 2);   // 插入到指定位置
list.remove(1);   // 删除元素
```

## 三、使用场景

| 数据结构 | 适用场景                                           |
| -------- | -------------------------------------------------- |
| 数组     | 需要**快速随机访问**、元素数量固定或可预估         |
| 链表     | **频繁插入删除**、不关心随机访问、内存连续性不重要 |

## String数组

## 面试总结一句话

> **Java 中 String 数组本质是存储 String 引用的固定长度数组，可以静态或动态初始化，访问和遍历效率高，但长度不可变，动态操作推荐使用集合。**

## 一、String 数组定义

```
String[] arr;        // 声明一个 String 数组（引用还未指向实际数组）
String arr2[];       // 另一种写法（等价）
```

- **本质**：数组存储的是 **对 String 对象的引用**，不是直接存放字符串内容
- 初始化前，数组元素默认值为 `null`

------

## 二、初始化方式

### 1️⃣ 静态初始化（直接赋值）

```
String[] arr = {"Java", "Python", "C++"};
```

### 2️⃣ 动态初始化（指定长度）

```
String[] arr = new String[3]; // 元素默认为 null
arr[0] = "Java";
arr[1] = "Python";
arr[2] = "C++";
```

### 3️⃣ 循环赋值

```
String[] arr = new String[5];
for (int i = 0; i < arr.length; i++) {
    arr[i] = "Item" + i;
}
```

------

## 三、常用操作

1. **遍历数组**

```
for (String s : arr) {
    System.out.println(s);
}
```

1. **获取长度**

```
System.out.println(arr.length);
```

1. **排序**

```
Arrays.sort(arr);
```

1. **查找 / 转 List**

```
List<String> list = Arrays.asList(arr);
```

------

## 四、特性与注意事项

1. **String 是不可变对象**

   - 数组中存储的是 String 对象的引用，不能改变 String 本身

2. **数组长度固定**

   - 一旦创建长度不可变
   - 动态添加元素需使用 `ArrayList<String>`

3. **多维数组**

   ```
   String[][] arr2D = { {"A","B"}, {"C","D"} };
   ```
